\# Machine Learning Projects



A curated collection of machine learning and data science projects implemented in Python.  

The repository demonstrates end-to-end workflows including data preprocessing, feature engineering, model training, evaluation, and visualization using real-world datasets.





\## Projects



\### Task 01 – Data Cleaning and Exploratory Data Analysis

Focused on preparing raw data for analysis and extracting insights through structured exploratory analysis.



Scope

\- Missing value treatment and duplicate removal

\- Data type correction and encoding

\- Outlier detection

\- Statistical and visual exploration of data



Tech Stack

\- Python, Pandas, NumPy

\- Matplotlib, Seaborn







\### Task 02 – House Price Prediction (Linear Regression)

A supervised learning project that predicts housing prices using regression techniques.



Scope

\- Data preprocessing and feature engineering

\- Categorical encoding and feature scaling

\- Linear Regression model training

\- Model evaluation using MSE and R²

\- Feature importance analysis



Tech Stack

\- Python, Pandas, NumPy

\- Scikit-learn

\- Matplotlib, Seaborn







\### Task 03 – Spam Email Classification

An NLP-based text classification system to identify spam messages.



Scope

\- Text preprocessing

\- TF-IDF feature extraction

\- Naive Bayes classification

\- Model evaluation using precision, recall, F1-score

\- Custom message prediction



Tech Stack

\- Python

\- Scikit-learn

\- NLP (TF-IDF)



\### Task 04 – Customer Churn Prediction

A classification project focused on predicting customer churn in a business context.



Scope

\- Data cleaning and preprocessing

\- Categorical feature encoding

\- Feature scaling

\- Logistic Regression with class imbalance handling

\- Evaluation using Accuracy, ROC-AUC, and Confusion Matrix



Tech Stack

\- Python, Pandas, NumPy

\- Scikit-learn

\- Seaborn
