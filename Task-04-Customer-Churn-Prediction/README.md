# Customer Churn Prediction using Machine Learning

This project predicts customer churn using a supervised machine learning classification model.

## Dataset
- Telco Customer Churn Dataset

## Approach
- Data cleaning and preprocessing
- Categorical encoding
- Feature scaling
- Logistic Regression model
- Evaluation using Accuracy, ROC-AUC, and Confusion Matrix

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Seaborn
- Logistic Regression

## Results
- Reliable churn prediction performance
- Balanced handling of class imbalance
- Business-relevant insights

## How to Run
1. Install dependencies from requirements.txt
2. Open `customer_churn.ipynb`
3. Run all cells
